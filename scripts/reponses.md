## Q2 — Scripts
1. Différence entre `bash hello.sh` et `./hello.sh` : 
bash hello.sh : lance le script en appelant directement Bash
./hello.sh : exécute le fichier comme un programme

2. À quoi sert le `#!` (shebang) ? Le shebang indique quel interpréteur doit être utilisé pour exécuter le script.
3. À quoi sert `chmod +x` ? Chmod +x ajoute le droit d’exécution à un fichier.
